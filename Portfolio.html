Comprehensive Analysis of Movie Ratings and Budget Trends

Conducted a thorough analysis of a dataset containing movie ratings and financial metrics to uncover patterns, relationships, and insights within the film industry.

  • Utilized Python and Pandas to import and preprocess a dataset of movie records, ensuring data integrity with zero nulls and duplicates.
  • Performed exploratory data analysis and computed descriptive statistics (mean, median, standard deviation) for numerical columns using Python, revealing key insights into film ratings and budgets.
  • Developed and implemented visualizations with Matplotlib and Seaborn, including scatter plots, pair plots, and heatmaps, to uncover patterns in film release trends, genre performance, and budget relationships.
  • Interpreted and presented findings from visualizations, identifying trends such as peak years for film releases and high-performing genres, and analyzed the budget impact on Rotten Tomatoes ratings, contributing to data-driven decision-making.

Employee Payroll Management Database Analysis

Designed and implemented a relational database schema in pgAdmin using SQL to manage employee information, payroll, and work locations.

  • Created and linked 12 tables to manage employee data, resolving integrity issues by establishing foreign key constraints.
  • Populated tables with data and developed SQL queries to analyze employee compensation, work hours, and state-wise distributions, providing actionable insights that improved payroll management and HR decision-making.
  • Presented key findings and recommendations for future projects to a non-technical audience, demonstrating effective communication of data analysis results and strategic planning.
